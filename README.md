# HEIC Viewer+

A lightweight desktop tool to view, crop, and convert modern image formats.

## Features
- Open HEIC / HEIF / AVIF / WebP / JPEG / PNG
- Crop images non-destructively
- Rotate & flip
- Convert between formats
- Save edited copies
- Remembers last opened folder

## Download
Go to **Releases** and download the Windows executable.

## Built With
- PySide6 (Qt)
- Pillow
- pillow-heif
